## Työn aihe
Työssä toteutan palvelun jääkiekko-otteluiden ja -pelaajien laukaisukarttojen esittämiseen. Käyttäjä voi tarkastella pelaajien ja otteluiden tarkempia tietoja, sekä asettaa pelaajia seurantaansa.
